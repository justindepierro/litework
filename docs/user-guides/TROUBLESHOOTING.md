# LiteWork Troubleshooting Guide

Common issues and their solutions for coaches and athletes using LiteWork.

## Account & Authentication Issues

### Can't Log In

**Symptoms**: Login fails with error message or wrong password.

**Solutions**:

1. **Verify Email and Password**:
   - Check for typos in email
   - Ensure Caps Lock is off
   - Try copying/pasting password

2. **Reset Your Password**:
   - Click "Forgot Password?" on login page
   - Enter your email address
   - Check email for reset link (check spam folder)
   - Click link and set new password
   - Try logging in again

3. **Clear Browser Cache**:
   - Chrome: Settings → Privacy → Clear browsing data
   - Safari: Preferences → Privacy → Manage Website Data
   - Firefox: Options → Privacy → Clear Data
   - Try logging in again

4. **Try Incognito/Private Mode**:
   - Open incognito/private browsing window
   - Attempt to log in
   - If successful, clear cache in normal browser

5. **Contact Support**:
   - If none of the above work
   - Email: support@litework.app
   - Include: Your email, browser type, error message

### Invitation Email Not Received

**Symptoms**: Coach/athlete sent invitation but no email received.

**Solutions**:

1. **Check Spam/Junk Folder**:
   - Look for emails from "noreply@litework.app"
   - Mark as "Not Spam" if found
   - Move to inbox

2. **Verify Email Address**:
   - Coach: Check athlete's email in their profile
   - Athlete: Confirm email with your coach
   - Look for typos (common: .com vs .co)

3. **Resend Invitation**:
   - Coach: Go to athlete's profile
   - Click "Resend Invitation"
   - Wait 5-10 minutes

4. **Check Email Filters**:
   - Look at email filtering rules
   - Whitelist "litework.app" domain
   - Check if emails are being forwarded elsewhere

5. **Use Alternative Email**:
   - Try different email address
   - Gmail, Outlook, or school email
   - Update profile after successful signup

### Session Expired / Logged Out Automatically

**Symptoms**: Logged out unexpectedly, "session expired" message.

**Solutions**:

1. **Log Back In**:
   - Sessions expire after 24 hours of inactivity
   - This is normal for security
   - Simply log in again

2. **Use "Remember Me"** (Coming Soon):
   - Stay logged in for 30 days
   - Only on trusted devices

3. **Check Browser Settings**:
   - Don't block cookies
   - Allow cookies for litework.app
   - Disable "Clear cookies on exit"

4. **Multiple Devices**:
   - Logging in on new device logs out old device
   - This is normal behavior
   - Log in on each device as needed

## Workout Issues

### Workout Not Showing for Athlete

**Symptoms**: Coach assigned workout but athlete can't see it.

**Solutions**:

1. **Check Start Date**:
   - Workouts only appear on/after start date
   - Coach: Verify start date in assignment
   - If start date is future, athlete won't see it yet

2. **Refresh the Page**:
   - Pull down to refresh (mobile)
   - Press F5 or Cmd+R (desktop)
   - Force quit and reopen app

3. **Check Assignment Status**:
   - Coach: Go to "Assignments" page
   - Verify assignment shows as "Active"
   - Check correct athlete was selected

4. **Verify Athlete Account**:
   - Ensure athlete accepted invitation
   - Athlete should have completed account setup
   - Check athlete appears in coach's athlete list

5. **Check Filters**:
   - Athlete: Check if filters are applied
   - Look in "All Workouts" tab
   - Clear any date or status filters

### Can't Start Workout (Live Mode)

**Symptoms**: "Start Workout" button doesn't work or workout won't load.

**Solutions**:

1. **Check Internet Connection**:
   - Workout data must load initially
   - Connect to Wi-Fi or cellular
   - Wait for workout to fully load

2. **Refresh the Workout**:
   - Go back to workout list
   - Click workout again
   - Wait for full load before starting

3. **Clear App Cache**:
   - Settings → Clear cache (in-app)
   - Or clear browser cache
   - Reopen workout

4. **Try Different Browser**:
   - Chrome, Safari, or Firefox
   - Some browsers work better than others
   - Consider using PWA version

5. **Check Workout Status**:
   - Ensure it's not already marked complete
   - If partially complete, use "Resume" instead of "Start"

### Sets Not Saving / Data Lost

**Symptoms**: Logged sets but they disappeared or didn't save.

**Solutions**:

1. **Check Internet Connection**:
   - Data syncs when online
   - Go online and wait for sync
   - Look for sync indicator

2. **Don't Close App Immediately**:
   - Wait 10-15 seconds after last set
   - Let data sync to server
   - Look for "Saved" confirmation

3. **Recover from Local Storage**:
   - Reopen the workout
   - Data may still be cached locally
   - Click "Resume Workout"

4. **Use Offline Mode Properly**:
   - Log sets offline is fine
   - Must connect to internet before closing
   - Data syncs automatically when online

5. **Submit Support Ticket**:
   - If data truly lost, contact support
   - Include: Date, workout name, approximate data
   - We may be able to recover from logs

### Exercise Not in Library

**Symptoms**: Can't find specific exercise when creating workout.

**Solutions**:

1. **Try Different Search Terms**:
   - "Bench Press" vs "Barbell Bench Press"
   - "Pullup" vs "Pull-up" vs "Pull Up"
   - Try muscle group filter

2. **Browse by Category**:
   - Click muscle group filter
   - Browse equipment type
   - Look through movement patterns

3. **Create Custom Exercise**:
   - Click "Add Exercise" → "Create Custom"
   - Enter exercise name and details
   - Add to your workout
   - Available for future workouts

4. **Request Exercise Addition**:
   - Email: support@litework.app
   - Include exercise name and description
   - We'll add to the library

5. **Use Similar Exercise**:
   - Find closest alternative
   - Add notes explaining substitution
   - Functionally equivalent

## Progress & History Issues

### Workout History Not Showing

**Symptoms**: Completed workouts but history is empty.

**Solutions**:

1. **Check Date Filter**:
   - Click "Filters"
   - Check date range
   - Try "All Time" or custom range

2. **Check Status Filter**:
   - Ensure "Completed" is selected
   - Try "All Status"
   - Clear all filters with "Reset"

3. **Verify Workout Was Completed**:
   - Go to workout
   - Check if marked "Complete"
   - May still be "In Progress"

4. **Check Other Account**:
   - If you have multiple accounts
   - Ensure you're logged into correct one
   - Coach vs athlete accounts are separate

5. **Refresh and Sync**:
   - Pull down to refresh
   - Connect to internet
   - Wait for sync to complete

### 1RM / PRs Not Updating

**Symptoms**: Set personal record but it's not showing in progress.

**Solutions**:

1. **Wait for Calculation**:
   - 1RMs calculated after workout completion
   - May take 5-10 minutes to update
   - Refresh progress page

2. **Check Exercise Match**:
   - Must be exact same exercise
   - "Barbell Squat" ≠ "Back Squat"
   - View exercise details to confirm

3. **Verify Rep Range**:
   - 1RMs only calculated from 1-12 rep sets
   - Higher rep sets (13+) not used for estimation
   - Must use sufficient weight

4. **Complete the Workout**:
   - PRs only count in completed workouts
   - In-progress workouts don't update PRs
   - Finish and save workout

5. **Manual Entry** (Coming Soon):
   - Add PRs manually
   - Enter tested 1RM values
   - Override automatic calculations

### CSV Export Not Working

**Symptoms**: Export button doesn't download file or file is empty.

**Solutions**:

1. **Check Browser Permissions**:
   - Allow downloads from litework.app
   - Check browser download settings
   - Look in Downloads folder

2. **Check Filter Results**:
   - Export only downloads filtered results
   - If no workouts match filters, file is empty
   - Clear filters to export all

3. **Try Different Browser**:
   - Chrome recommended for exports
   - Safari sometimes blocks downloads
   - Firefox works well too

4. **Disable Popup Blocker**:
   - Popup blockers may interfere
   - Whitelist litework.app
   - Try again

5. **Use Desktop**:
   - Exports work better on desktop
   - Mobile browsers may have issues
   - Transfer to computer if needed

## Mobile / PWA Issues

### Can't Install PWA

**Symptoms**: "Add to Home Screen" option not appearing.

**Solutions**:

**iPhone (iOS)**:

1. Must use Safari (not Chrome or Firefox)
2. Tap Share button (square with arrow up)
3. Scroll down to "Add to Home Screen"
4. If missing, update iOS to latest version

**Android**:

1. Use Chrome browser
2. Tap menu (three dots)
3. Look for "Add to Home Screen" or "Install App"
4. If missing, update Chrome to latest version

**Desktop**:

1. Use Chrome or Edge
2. Look for install icon in address bar
3. Click "Install"
4. If missing, visit litework.app in Chrome

### PWA Not Working Offline

**Symptoms**: Installed PWA but can't use offline.

**Solutions**:

1. **Initial Load Required**:
   - Must open workout while online first
   - Data caches for offline use
   - Can't download new workouts offline

2. **Limited Offline Features**:
   - View workouts: Yes
   - Log sets: Yes
   - Create workouts: No
   - Assign workouts: No
   - View new assignments: No

3. **Sync When Back Online**:
   - Connect to Wi-Fi or cellular
   - Open app
   - Wait for sync indicator
   - Data uploads automatically

4. **Reinstall PWA**:
   - Delete from home screen
   - Clear cache
   - Reinstall from browser
   - Open while online

### App Running Slowly

**Symptoms**: Pages loading slowly, buttons laggy, general slowness.

**Solutions**:

1. **Clear Cache**:
   - In-app: Settings → Clear Cache
   - Browser: Clear browsing data
   - Improves performance significantly

2. **Close Other Apps**:
   - Free up device memory
   - Close unused browser tabs
   - Restart device if very slow

3. **Update App/Browser**:
   - Update to latest version
   - Update operating system
   - Restart after updating

4. **Check Internet Speed**:
   - Test connection speed
   - Try Wi-Fi vs cellular
   - Move closer to router

5. **Reinstall PWA**:
   - Delete and reinstall
   - Often fixes performance issues
   - Don't worry, data is saved on server

## Coach-Specific Issues

### Can't Assign Workout to Group

**Symptoms**: Assignment fails or athletes don't receive workout.

**Solutions**:

1. **Verify Group Has Athletes**:
   - Go to Groups page
   - Open group
   - Check members list isn't empty

2. **Check Group Status**:
   - Ensure group is active
   - Archived groups can't receive assignments
   - Reactivate if needed

3. **Verify Workout Exists**:
   - Workout must be created first
   - Can't assign deleted workouts
   - Check workout isn't archived

4. **Check Start Date**:
   - Can't set start date in past
   - Athletes won't see workouts until start date
   - Verify date is correct

5. **Try Individual Assignment First**:
   - Test with one athlete
   - If works, group assignment should too
   - May be temporary server issue

### Athletes Not Completing Workouts

**Symptoms**: Assigned workouts but low completion rate.

**Solutions**:

1. **Check Workout Difficulty**:
   - May be too hard/long
   - Ask athletes for feedback
   - Adjust volume or intensity

2. **Verify Athletes Understand App**:
   - Share athlete quick start guide
   - Demo in person
   - Create video tutorial

3. **Check Assignment Visibility**:
   - Ensure athletes can see workouts
   - Test with your own athlete account
   - Verify start dates are correct

4. **Improve Communication**:
   - Add notes to assignments
   - Explain workout purpose
   - Motivate through messaging

5. **Simplify Workouts**:
   - Start with simple 4-6 exercise workouts
   - Gradually add complexity
   - Ensure exercises are familiar

### Can't Modify Group Assignment for Individual

**Symptoms**: Trying to customize workout for one athlete but changes affect everyone.

**Solutions**:

1. **Use Individual Modification Feature**:
   - Go to Assignments page
   - Find group assignment
   - Click "Modify for Athlete"
   - Select specific athlete
   - Changes only affect that athlete

2. **Don't Edit Original Workout**:
   - Editing the base workout affects all assignments
   - Use individual modification instead
   - Original workout stays the same for others

3. **Create Separate Assignment**:
   - Alternative: Assign different workout to individual
   - Won't show as group assignment
   - More work but gives full control

## Error Messages

### "Rate Limit Exceeded"

**Meaning**: Too many requests in short time.

**Solutions**:

- Wait 5-10 minutes
- Don't spam buttons
- Normal usage rarely triggers this

### "Session Expired"

**Meaning**: Logged in too long ago, session timed out.

**Solutions**:

- Log in again
- Sessions last 24 hours
- This is normal security measure

### "Network Error"

**Meaning**: Can't connect to server.

**Solutions**:

- Check internet connection
- Try different network
- Server may be down (rare)
- Wait a few minutes and retry

### "Permission Denied"

**Meaning**: Trying to access something you don't have permission for.

**Solutions**:

- Verify you're logged in as correct role
- Athletes can't access coach features
- Contact admin if you need different permissions

### "Invalid Data"

**Meaning**: Form input doesn't meet requirements.

**Solutions**:

- Check all required fields are filled
- Verify email format
- Ensure numbers are in valid ranges
- Read field help text

## Performance Issues

### Slow Loading

**Symptoms**: Pages take long to load, spinning indicators.

**Solutions**:

1. **Check Internet Speed**:
   - Run speed test
   - Minimum 3 Mbps recommended
   - Use Wi-Fi when possible

2. **Clear Cache**:
   - Often fixes slow loading
   - Clear browser cache
   - Restart browser

3. **Reduce Extensions**:
   - Disable ad blockers for litework.app
   - Too many extensions slow browser
   - Use incognito to test

4. **Update Browser**:
   - Old browsers are slower
   - Update to latest version
   - Consider switching browsers

### High Data Usage

**Symptoms**: App using too much mobile data.

**Solutions**:

1. **Use Wi-Fi**:
   - Connect to Wi-Fi when available
   - Download workouts on Wi-Fi
   - Sync data on Wi-Fi

2. **Limit Video Viewing**:
   - Exercise videos use most data
   - Don't autoplay videos
   - Watch on Wi-Fi only

3. **Reduce Sync Frequency**:
   - Don't refresh constantly
   - Sync happens automatically
   - Manual refresh uses data

## Getting Additional Help

### Before Contacting Support

Gather this information:

- What were you trying to do?
- What happened instead?
- Error messages (screenshot if possible)
- Browser and version (Chrome 95, Safari 15, etc.)
- Device (iPhone 12, Windows PC, etc.)
- Your email address
- Steps to reproduce the problem

### Contact Methods

**Email Support**:

- support@litework.app
- Response within 24 hours (weekdays)
- Include all information above
- Attach screenshots if helpful

**Emergency Issues**:

- For critical bugs only
- Include "URGENT" in subject
- Describe impact on your training

### Feature Requests

- Email: support@litework.app
- Subject: "Feature Request"
- Describe desired feature
- Explain use case
- We review all requests for future releases

### Bug Reports

- Email: support@litework.app
- Subject: "Bug Report"
- Detailed steps to reproduce
- Expected vs actual behavior
- Screenshots or screen recording
- Browser console errors (if technical)
