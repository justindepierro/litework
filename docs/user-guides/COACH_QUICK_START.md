# Coach Quick Start Guide

Welcome to LiteWork! This guide will help you get started with managing your athletes and creating effective workout programs.

## Getting Started

### 1. First Login

After receiving your invitation email:

1. Click the invitation link
2. Create your password (minimum 8 characters)
3. You'll be taken to the coach dashboard

### 2. Understanding Your Dashboard

Your dashboard shows:

- **Athletes**: Total number of athletes you're coaching
- **Active Workouts**: Workouts currently assigned
- **Pending Assignments**: Workouts waiting to be started
- **Recent Activity**: Latest athlete workout completions

## Managing Athletes

### Adding Athletes

1. Click **"Athletes"** in the navigation menu
2. Click **"Add Athlete"** button
3. Fill in athlete information:
   - First and Last Name
   - Email address
   - Sport (Football, Volleyball, etc.)
   - Position (if applicable)
4. Click **"Send Invitation"**
5. The athlete will receive an email to create their account

### Organizing Athletes into Groups

Groups help you assign workouts to multiple athletes at once.

**To Create a Group:**

1. Go to **"Groups"** page
2. Click **"Create Group"**
3. Enter group details:
   - Group Name (e.g., "Varsity Football", "JV Volleyball")
   - Sport
   - Description (optional)
4. Select athletes to add to the group
5. Click **"Create Group"**

**Best Practices:**

- Create groups by sport and experience level
- Use position-specific groups for sports like football
- Keep groups between 10-30 athletes for easy management

## Creating Workouts

### Using the Workout Editor

1. Navigate to **"Workouts"** → **"Create Workout"**
2. Enter workout details:
   - **Name**: Be specific (e.g., "Upper Body Power - Week 1")
   - **Description**: Include focus areas or coaching points
   - **Duration**: Estimated time (minutes)

### Adding Exercises

**To Add a Single Exercise:**

1. Click **"Add Exercise"**
2. Search for the exercise in the library (500+ exercises available)
3. Set parameters:
   - **Sets**: Number of sets (e.g., 3-5)
   - **Reps**: Reps per set (e.g., 8-12)
   - **Weight**: Target weight or percentage of 1RM
   - **Tempo**: Optional (e.g., "3-1-1-0" for controlled eccentric)
   - **Rest**: Rest between sets (seconds)
   - **Notes**: Exercise-specific coaching cues

**Exercise Parameters Explained:**

- **Sets**: Total number of times to perform the exercise
- **Reps**: Repetitions per set (can be a range like "8-12")
- **Weight**: Can be specific (135 lbs) or percentage (70% 1RM)
- **Tempo**: Four numbers representing eccentric-pause-concentric-pause in seconds
- **RPE**: Rate of Perceived Exertion (1-10 scale)
- **Rest**: Rest period between sets in seconds

### Creating Exercise Groups

LiteWork supports three types of exercise grouping:

#### 1. Supersets (2-4 exercises back-to-back)

Perfect for:

- Antagonistic muscle pairings (push/pull)
- Time-efficient training
- Increased training density

**To Create a Superset:**

1. Click **"Create Group"** → **"Superset"**
2. Add 2-4 exercises
3. Minimal rest between exercises
4. Rest after completing all exercises

**Example:**

- A1. Bench Press - 4x8
- A2. Barbell Row - 4x8
- Rest 90 seconds, repeat

#### 2. Circuits (5+ exercises)

Perfect for:

- Conditioning work
- Metabolic training
- Large groups with limited equipment

**To Create a Circuit:**

1. Click **"Create Group"** → **"Circuit"**
2. Add 5+ exercises
3. Set rounds (usually 2-4)
4. Set rest between exercises and rounds

**Example:**

- Push-ups x 15
- Squats x 20
- Pull-ups x 10
- Plank x 45 seconds
- Rest 2 minutes, repeat for 3 rounds

#### 3. Sections (Organizing workout phases)

Perfect for:

- Warm-up / Main Lift / Accessory / Cooldown
- Organizing complex workouts
- Clear visual separation

**To Create a Section:**

1. Click **"Create Section"**
2. Name the section (e.g., "Warm-up", "Primary Lifts")
3. Add exercises within the section

### Using Workout Templates

Save time by creating reusable templates:

1. Click **"Save as Template"** when creating a workout
2. Name your template (e.g., "Upper Body Template")
3. Access templates from **"Templates"** tab
4. Click **"Use Template"** to create a new workout from it

## Assigning Workouts

### Individual Assignment

1. Go to **"Workouts"** page
2. Find the workout you want to assign
3. Click the three-dot menu → **"Assign"**
4. Select individual athlete(s)
5. Set the start date
6. Click **"Assign Workout"**

### Group Assignment

1. From the workout's menu, click **"Assign to Group"**
2. Select the group
3. Set the start date
4. Click **"Assign to Group"**

### Assignment Scheduling

You can schedule assignments in advance:

- **Start Date**: When the workout becomes available
- **Due Date**: Optional deadline
- **Recurrence**: For ongoing programs (coming soon)

### Individual Modifications

After assigning a workout to a group, you can customize it for specific athletes:

1. Go to **"Assignments"** page
2. Find the group assignment
3. Click **"Modify for Athlete"**
4. Select the athlete
5. Make changes:
   - Change exercise weights
   - Modify sets/reps
   - Substitute exercises
   - Add notes
6. Save modifications

**Common Use Cases:**

- Injured athletes needing alternative exercises
- Beginners needing reduced volume
- Advanced athletes needing increased intensity

## Exercise Library

### Browsing the Library

1. Navigate to **"Exercises"**
2. Filter by:
   - Muscle group (Chest, Back, Legs, etc.)
   - Equipment (Barbell, Dumbbell, Bodyweight)
   - Movement pattern (Push, Pull, Squat, Hinge)
3. Click an exercise to view:
   - Detailed instructions
   - Video demonstrations (if available)

### Adding Custom Exercises

Create sport-specific or custom exercises:

1. Go to **"Exercises"** → **"Add Exercise"**
2. Fill in details:
   - Exercise name
   - Description
   - Primary and secondary muscle groups
   - Equipment required
   - Video URL (YouTube/Vimeo)
3. Click **"Create Exercise"**

### Exercise Favorites

Mark frequently used exercises as favorites for quick access:

1. Click the star icon on any exercise
2. Access favorites from the **"Favorites"** filter

## Best Practices

### Workout Design

- **Start Simple**: Begin with 4-6 exercises per workout
- **Progressive Overload**: Gradually increase volume or intensity weekly
- **Recovery**: Include rest days and deload weeks
- **Specificity**: Design workouts for the athlete's sport and goals

### Communication

- **Exercise Notes**: Add coaching cues in the exercise notes field
- **Workout Description**: Include the workout's purpose and focus areas
- **Assignment Notes**: Add specific instructions when assigning

### Organization

- **Naming Convention**: Use consistent workout naming (e.g., "Upper - Week 1 - Day 1")
- **Templates**: Create templates for repeating programs
- **Groups**: Organize athletes by sport, position, and experience level
- **Tags**: Use descriptive tags for easy searching (coming soon)

### Monitoring

- **Check-ins**: Review athlete workouts weekly
- **Adjust**: Modify programs based on completion rates and feedback
- **Encourage**: Provide feedback through session notes
- **Track**: Monitor long-term trends in strength and volume

## Common Workflows

### Creating a 4-Week Program

1. Plan your program structure:
   - Week 1: Adaptation (60-70% intensity)
   - Week 2: Build (70-80% intensity)
   - Week 3: Peak (80-90% intensity)
   - Week 4: Deload (50-60% intensity)

2. Create workouts for Week 1
3. Use **"Duplicate"** to create Weeks 2-4
4. Adjust weights/volume for each week
5. Save each week's workouts as templates
6. Assign Week 1, then assign subsequent weeks as needed

### Managing Pre-Season Training

1. Create athlete groups by team
2. Design sport-specific workouts:
   - Strength & power focus
   - Sport-specific movements
   - Conditioning work
3. Assign to groups with 2-3 week planning window
4. Monitor completion and adjust volume
5. Provide feedback on quality

### In-Season Maintenance

1. Reduce volume (2-3 days per week)
2. Focus on primary lifts
3. Use shorter, efficient workouts (30-45 minutes)
4. Monitor fatigue through RPE ratings
5. Adjust based on game schedule

## Troubleshooting

### Athletes Not Receiving Invitations

- Check spam/junk folders
- Verify email address is correct
- Resend invitation from athlete's profile
- Contact support if issues persist

### Workouts Not Appearing for Athletes

- Verify assignment was completed
- Check start date is today or earlier
- Ensure athlete is active in the system
- Refresh the athlete's app

### Exercise Not in Library

- Search for alternative names (e.g., "bench press" vs "barbell bench press")
- Create a custom exercise
- Request addition through support

### Group Assignment Issues

- Verify athletes are in the selected group
- Check for individual modifications that may override
- Ensure group is active
