# Exercise Configuration - Quick Reference Card

## ‚ö° Quick Field Guide

### Basic Fields (Required)

- **Exercise Name**: Name of the movement
- **Sets**: Number of sets to perform
- **Reps**: Repetitions per set

### Tempo (Optional)

**Format**: `Down-Pause-Up-Top` (e.g., `3-1-1-0`)

| Tempo     | Use Case                     |
| --------- | ---------------------------- |
| `2-0-2-0` | Standard controlled tempo    |
| `3-1-1-0` | Strength emphasis            |
| `4-0-1-0` | Hypertrophy (slow eccentric) |
| `1-0-X-0` | Power/explosive              |

### Load Type (Required)

| Type             | When to Use                  | Example  |
| ---------------- | ---------------------------- | -------- |
| **Fixed Weight** | Specific weight prescription | 185 lbs  |
| **% of 1RM**     | Auto-regulated programming   | 80%      |
| **Bodyweight**   | No external load             | Pull-ups |

### Each Side (Optional)

‚òê **Uncheck** for bilateral exercises (squat, bench press)  
‚òë **Check** for unilateral exercises (Bulgarian split squat, single-arm row)

**When checked**: Reps are performed on EACH side (total reps √ó 2)

**Applies to**: Legs, arms, or any single-side exercise

### Rest Time (Optional)

| Time     | Use Case                 |
| -------- | ------------------------ |
| 30-45s   | Circuits, conditioning   |
| 60-90s   | Accessories, hypertrophy |
| 120-180s | Main compound lifts      |
| 240-300s | Heavy maximal efforts    |

### Notes (Optional)

Add specific instructions:

- Equipment: "Use resistance band"
- Cues: "Focus on squeeze at top"
- Setup: "Bench at 30-degree incline"
- Safety: "Use spotter on heavy sets"

## üéØ Common Patterns

### Main Compound Lifts

- Sets: 4-5
- Reps: 3-6
- Tempo: 2-1-1-0 or 3-1-1-0
- Load: 80-90% of 1RM
- Rest: 180-300 seconds

### Hypertrophy Accessories

- Sets: 3-4
- Reps: 8-12
- Tempo: 3-0-2-0 or 3-1-1-0
- Load: 60-75% or Fixed
- Rest: 60-90 seconds

### Unilateral Exercises

- ‚òë Check "Each Leg"
- Reps: 8-12 per leg
- Rest: 60-90 seconds
- Fixed weight recommended

### Conditioning Circuits

- Sets: 3 (rounds)
- Reps: 10-15
- Tempo: Standard (2-0-2-0)
- Load: Bodyweight or light
- Rest: 30-45 seconds

## üí° Pro Tips

**Tempo**:

- Always control the eccentric
- Pauses build strength
- Explosive = "X" in notation

**Each Leg**:

- Start with weaker side
- Same reps each leg
- Track imbalances in notes

**Load**:

- Use % for main lifts
- Fixed for accessories
- Bodyweight for warm-ups

**Notes**:

- Keep it concise
- Bullet points work best
- Update as needed
