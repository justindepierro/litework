# Workout Blocks - Quick Reference Card

## What Are Workout Blocks?

**Workout Blocks** are reusable exercise templates you can quickly insert into your workouts. Think of them as pre-built LEGO pieces for your training programs.

## Quick Start Guide

### Building a Workout with Blocks

1. **Create/Edit Workout** ‚Üí Click "Create Workout" or edit existing
2. **Add Block** ‚Üí Click the purple "Add Block" button
3. **Browse** ‚Üí Filter by category or search
4. **Select** ‚Üí Click a block to insert it
5. **Customize** ‚Üí Edit exercises as needed
6. **Save** ‚Üí Done!

**Time Saved**: Build a complete workout in under 60 seconds! ‚ö°

## FAQ

**Q: Can I edit a block after inserting it?**  
A: Yes! Once inserted, exercises become part of your workout and can be freely edited.

**Q: If I edit an inserted block, does it change the original?**  
A: No! Inserted exercises are independent. The original block remains unchanged.

**Q: Can I create my own blocks?**  
A: Coming soon! Phase 2 will add "Save as Block" functionality.

**Q: What happens if I use the same block in multiple workouts?**  
A: Each workout gets its own copy. Edit one without affecting others.

**Q: How do I know which blocks are most popular?**  
A: Blocks show usage count. Sort by "Most Used" to see top performers.
